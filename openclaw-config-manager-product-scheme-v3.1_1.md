## 📋 产品概述

**ClawGuard** 是 OpenClaw 的配置管理工具，提供配置文件备份、恢复、迁移和工作区管理功能。

**核心价值：**
- 防止配置丢失（意外删除、误操作）
- 快速环境迁移（多设备同步）
- 配置版本管理（回滚、对比）
- 工作区自动备份（保护重要数据）

## 💎 核心功能

### 1. CLI 命令行（开发者专用）

#### 1.1 配置备份
**分层备份策略：**
- **Level 1: 配置文件** - 只备份 `~/.openclaw/` 下的配置文件
- **Level 2: 配置 + 系统文件** - 配置文件 + SOUL.md、USER.md、MEMORY.md 等
- **Level 3: 完整工作区** - 备份整个 `~/.openclaw/workspace/` 目录

**命令：**
```bash
# 只备份配置文件（Level 1）
clawguard backup --level config

# 备份配置 + 系统文件（Level 2）
clawguard backup --level system

# 备份完整工作区（Level 3）
clawguard backup --level full

# 指定备份名称
clawguard backup --level config --name "生产环境配置"

# 备份到指定目录
clawguard backup --level full --output ~/backups
```

#### 1.2 配置恢复
**命令：**
```bash
# 列出所有备份
clawguard backup list

# 恢复最新备份
clawguard restore

# 恢复指定备份
clawguard restore <backup-id>

# 预览备份内容
clawguard restore <backup-id> --dry-run
```

#### 1.3 配置迁移
**分层导出策略：**
- **Level 1: 配置文件** - 只导出 `~/.openclaw/` 下的配置文件
- **Level 2: 配置 + 系统文件** - 配置文件 + SOUL.md、USER.md、MEMORY.md 等
- **Level 3: 完整工作区** - 导出整个 `~/.openclaw/workspace/` 目录

**命令：**
```bash
# 只导出配置文件（Level 1）→ 导出为 config.json
clawguard export --level config

# 导出配置 + 系统文件（Level 2）→ 导出为 config-system.tar.gz
clawguard export --level system

# 导出完整工作区（Level 3）→ 导出为 workspace.tar.gz
clawguard export --level full

# 指定导出目录（默认 ~/.clawguard/exports/）
clawguard export --level config --output ~/exports

# 加密导出
clawguard export --level config --encrypt

# 导入配置（JSON 或 TAR.GZ）
clawguard import config.json

# 导入加密配置
clawguard import config.enc --decrypt
```

#### 1.4 配置对比
**命令：**
```bash
# 对比当前配置和备份
clawguard diff <backup-id>

# 对比两个备份
clawguard diff <backup-id-1> <backup-id-2>

# 忽略特定字段
clawguard diff <backup-id-1> <backup-id-2> --ignore "token,secret"
```

### 4. 定时任务管理

#### 4.1 自动备份
**功能：**
- 定时自动备份配置和工作区
- 保留策略（保留最近 N 个备份）
- 备份通知（成功/失败）

**命令（CLI）：**
```bash
# 启用自动备份（每天 22:00）
clawguard schedule enable --cron "0 22 * * *"

# 配置保留策略（保留最近 7 天）
clawguard schedule retain --days 7

# 查看定时任务
clawguard schedule list

# 禁用自动备份
clawguard schedule disable
```

**UI 功能（Web UI）：**
- 定时任务配置面板
- Cron 表达式编辑器
- 保留策略设置
- 任务执行日志

### 5. 安全功能

#### 5.1 备份加密
**功能：**
- 备份文件 AES-256 加密

- 加密元数据（算法、盐、IV）

**命令（CLI）：**
```bash
# 设置加密密钥
clawguard security set-key

# 加密备份
clawguard backup --encrypt

# 解密备份
clawguard backup <backup-id> --decrypt
```

**UI 功能（Web UI）：**
- 密钥设置界面

**UI 功能（Web UI）：**
- 备份验证按钮
